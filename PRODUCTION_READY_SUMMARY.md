Clear browser cache and try again.\n\n**Q: Emails not sending?**  \nA: Verify mail() function is enabled on server. Check SMTP configuration.\n\n**Q: Mobile layout broken?**  \nA: Clear browser cache. Check that CSS media queries are loading.\n\n**Q: CSRF token errors?**  \nA: Ensure session_start() is called before any output.
