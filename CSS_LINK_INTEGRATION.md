# CSS Link Integration - Exact Code for Each File

## 📝 COPY & PASTE INSTRUCTIONS

Add this line to the `<head>` section of each PHP file, right after the Bootstrap CSS link.

---

## 1️⃣ index.php

**Find this section:**
```html
    <link rel="stylesheet" href="css/bootstrap.min.css">
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.1/css/all.min.css"
```

**Add this line after bootstrap.min.css:**
```html
    <link rel="stylesheet" href="css/responsive-master-fix.css">
```

**Result should look like:**
```html
    <link rel="stylesheet" href="css/bootstrap.min.css">
    <link rel="stylesheet" href="css/responsive-master-fix.css">
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.1/css/all.min.css"
```
